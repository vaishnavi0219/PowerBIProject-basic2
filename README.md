## Power BI Project 2: Insurance Data Analysis

This project focuses on analyzing insurance data using Power BI, covering SQL integration, DAX, visualization, deployment, and security.

---

### 📊 Objective
To analyze insurance data, track key performance metrics, and provide insights using interactive dashboards.

---

### ⚙️ Data Source
- SQL Server (MSSQL)

---

### 🔧 Data Preparation
- Imported data into SQL Server
- Connected Power BI to SQL Server
- Performed data profiling and cleaning
- Structured dataset for reporting

---

### 🧠 DAX Measures
- KPIs for insurance performance
- Aggregations and ratio calculations
- Functions used:
  - CALCULATE
  - SUM
  - COUNT
  - DIVIDE

---

### 📈 Visualizations
- KPI Cards
- Bar & Line Charts (trend analysis)
- Donut Chart (distribution)
- Matrix Visual (detailed breakdown)
- Ribbon Chart
- Multi-row Cards

---

### 🔍 Features
- Drill-through analysis for detailed insights
- Interactive slicers and filters
- Custom filter interactions

---

### 🔐 Security (RLS)
- Implemented Row-Level Security in Power BI Desktop
- Deployed and tested RLS in Power BI Service

---

### 🔄 Deployment
- Published report to Power BI Service
- Configured scheduled refresh
- Tested data refresh functionality

---

### 🧪 Advanced Analytics
- Sentiment analysis using Power Query
- Integrated sentiment insights into dashboard

---

### ✅ Validation
- Verified KPI calculations
- Ensured consistency across visuals

---

### 🚀 Outcome
- Delivered actionable insights on insurance data
- Built secure and scalable BI solution

---

### 🛠 Tools Used
- Power BI Desktop & Service
- SQL Server (MSSQL)
- Power Query
- DAX
